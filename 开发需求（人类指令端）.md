## 项目背景

本项目起源于与艺术家范文南合作的一个硬核科幻设定——设计一艘千人万吨级的木星-海王星往返转移飞行器。为追求科学严谨性，我们结合轨道动力学专业知识，完成了飞船往返弹道的优化设计。本项目旨在将此前已开发的核心轨道动力学优化代码，转化为一款简洁易用的工具，服务于科普教育与可视化演示。

本项目涉及航天、轨道动力学及数学等领域的基础知识，主要包括：

1. 轨道状态描述：位置、速度、加速度的基本概念
2. 限制性二体问题
3. 速度增量（ΔV）概念
4. 脉冲转移与连续推力转移
5.  porkchop 猪排图概念

此外，本项目还将作为课程展示内容，重点阐述直接配点法（Direct Collocation）的核心思想及其在轨道优化问题中的应用。

因此，本项目采用"一核三用"的模式：同一核心代码支撑科幻设定验证、SOLO挑战赛参与、课程教学展示三大应用场景。

## 项目开发功能要求（初始人类指令）

1. 现有代码位于 `DirectCollocation` 文件夹下，请先阅读并理解框架内容。
2. 深入理解框架中的三个核心 Python 文件（`PlotResults3D.py`、`SolarSystemModel.py`、`TrajectoryCollocation3D.py`），并在新框架下进行重构，实现项目功能。
3. 项目愿景：基于上述工作，开发一套专注于星际转移弹道优化的太阳系内轨道设计工具，具体包括：
    1. **3D 可视化太阳系场景**：支持拖拽、缩放、平移视角，多角度观察行星位置；
    2. **轨道规划功能**：用户可选择出发行星、出发时刻及期望飞行时间，自动生成至目标行星的转移轨道，并提供可视化展示与 ΔV 计算结果；
    3. 【拓展】**太阳距离约束**：在原有基础上增加与太阳的最小距离约束，优化近太阳段飞行安全性。
    4. 【拓展】**推力约束**：在原有基础上增加推力加速度约束，若推力需求过大，提示弹道不可达，并给出所需推力加速度。
    5. 【拓展】**porkchop 图绘制**：基于上述功能，支持给定时间范围内的发射窗口分析图生成。
    6. 【拓展】**出发方向优化**：优化航天器相对于出发行星及目标行星的初始/终端方向角，提升转移效率。
4. 对于开发的平台，考虑使用python的tkinter库，实现简单的用户界面，若时间充足，考虑使用html的网页界面。

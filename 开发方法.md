## 9. 附录

### 9.1 核心算法说明

#### 直接配点法
直接配点法是一种数值优化方法，用于求解最优控制问题。在轨道优化中，它将连续的轨迹离散化为有限个点，通过优化这些点的状态和控制输入，来最小化目标函数（如燃料消耗）。

#### 猪排图
猪排图（Porkchop Plot）是一种用于分析发射窗口的工具，它展示了不同出发时间和飞行时间组合下的速度增量（ΔV），帮助确定最优的发射窗口。

### 9.2 关键参数说明

| 参数 | 说明 | 单位 | 范围 |
|------|------|------|------|
| start_year | 出发年份 | 年 | 2000-2500 |
| start_month | 出发月份 | 月 | 1-12 |
| start_day | 出发日期 | 日 | 1-31 |
| tof_years | 飞行时间 | 年 | 0.5-10 |
| departure_body | 出发星体 | - | 太阳系行星 |
| arrival_body | 到达星体 | - | 太阳系行星 |
| N | 网格数 | - | 10-200 |
| rbound_factor | 太阳距离约束因子 | - | 0.1-1.0 |
| thrust_limit | 推力加速度限制 | km/s² | 0.001-1.0 |

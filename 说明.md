# 直接配点法轨迹优化说明

## 1. 直接配点法原理

直接配点法（Direct Collocation）是一种将连续最优控制问题转换为非线性规划问题的数值方法。该方法通过在时间域上离散化状态和控制变量，将微分方程约束转换为代数约束，从而可以使用标准的优化算法求解。

### 1.1 数学基础

#### 状态空间模型
考虑航天器在中心天体引力场中的运动，其状态空间模型为：

$$
\dot{\mathbf{r}} = \mathbf{v}
$$

$$
\dot{\mathbf{v}} = -\frac{\mu}{r^3}\mathbf{r} + \mathbf{u}
$$

其中：
- \(\mathbf{r} = [x, y, z]^T\) 是航天器的位置向量
- \(\mathbf{v} = [v_x, v_y, v_z]^T\) 是航天器的速度向量
- \(\mu\) 是中心天体的引力常数
- \(r = \|\mathbf{r}\|\) 是航天器到中心天体的距离
- \(\mathbf{u} = [u_x, u_y, u_z]^T\) 是控制输入（推力加速度）

#### 优化目标
我们的目标是最小化燃料消耗，通常通过最小化控制输入的平方积分来实现：

```math
J = \int_{t_0}^{t_f} \frac{1}{2}\|\mathbf{u}(t)\|^2 dt
```

### 1.2 梯形配点法

梯形配点法通过以下步骤离散化连续问题：

1. **时间离散化**：将时间区间 \([t_0, t_f]\) 划分为 \(N\) 个等长区间，时间步长为 \(dt = (t_f - t_0)/N\)

2. **状态和控制离散化**：在每个时间点 \(t_k = t_0 + k \cdot dt\) 处定义状态变量 \(\mathbf{r}_k\) 和 \(\mathbf{v}_k\)，以及控制变量 \(\mathbf{u}_k\)

3. **动力学约束**：使用梯形积分规则近似微分方程：

   ```math
   \mathbf{r}_{k+1} = \mathbf{r}_k + \frac{dt}{2}(\mathbf{v}_k + \mathbf{v}_{k+1})
   ```

   ```math
   \mathbf{v}_{k+1} = \mathbf{v}_k + \frac{dt}{2}(\mathbf{a}_k + \mathbf{a}_{k+1})
   ```

   其中 \(\mathbf{a}_k = -\frac{\mu}{r_k^3}\mathbf{r}_k + \mathbf{u}_k\) 是加速度

4. **边界约束**：满足初始和终端条件：
   - \(\mathbf{r}_0 = \mathbf{r}(t_0)\)
   - \(\mathbf{v}_0 = \mathbf{v}(t_0)\)
   - \(\mathbf{r}_N = \mathbf{r}(t_f)\)
   - \(\mathbf{v}_N = \mathbf{v}(t_f)\)

5. **路径约束**：如最小距离约束、最大推力约束等

## 2. 代码实现

### 2.1 核心实现

在 `core/trajectory_optimizer.py` 文件中，`trapezoidal_collocation_3d` 方法实现了梯形配点法：

#### 目标函数
```python
def objective(states):
    ux = states[6*N:7*N]
    uy = states[7*N:8*N]
    uz = states[8*N:9*N]
    u = np.sqrt(ux**2 + uy**2 + uz**2)  # 控制输入的大小

    J = 0
    for k in range(N - 1):
        J += 0.5 * (u[k] ** 2 + u[k + 1] ** 2) * dt
    return J
```

#### 动力学约束
```python
def equ_constraints(states, s0, sf, dt):
    # 提取状态变量
    x_pos = states[:N]
    y_pos = states[N:2*N]
    z_pos = states[2*N:3*N]
    vx = states[3*N:4*N]
    vy = states[4*N:5*N]
    vz = states[5*N:6*N]
    u_x = states[6*N:7*N]
    u_y = states[7*N:8*N]
    u_z = states[8*N:9*N]

    # 计算加速度
    r = np.sqrt(x_pos**2 + y_pos**2 + z_pos**2)
    a_x = (-mu/r**3) * x_pos + u_x
    a_y = (-mu/r**3) * y_pos + u_y
    a_z = (-mu/r**3) * z_pos + u_z

    cons = []

    # 动力学约束
    for k in range(N - 1):
        cons.append(x_pos[k + 1] - x_pos[k] - 0.5 * dt * (vx[k + 1] + vx[k]))
        cons.append(y_pos[k + 1] - y_pos[k] - 0.5 * dt * (vy[k + 1] + vy[k]))
        cons.append(z_pos[k + 1] - z_pos[k] - 0.5 * dt * (vz[k + 1] + vz[k]))
        cons.append(vx[k + 1] - vx[k] - 0.5 * dt * (a_x[k + 1] + a_x[k]))
        cons.append(vy[k + 1] - vy[k] - 0.5 * dt * (a_y[k + 1] + a_y[k]))
        cons.append(vz[k + 1] - vz[k] - 0.5 * dt * (a_z[k + 1] + a_z[k]))

    # 边界约束
    cons.append(x_pos[0] - x0)
    cons.append(y_pos[0] - y0)
    cons.append(z_pos[0] - z0)
    cons.append(vx[0] - vx0)
    cons.append(vy[0] - vy0)
    cons.append(vz[0] - vz0)

    cons.append(x_pos[-1] - xf)
    cons.append(y_pos[-1] - yf)
    cons.append(z_pos[-1] - zf)
    cons.append(vx[-1] - vxf)
    cons.append(vy[-1] - vyf)
    cons.append(vz[-1] - vzf)

    return cons
```
